# Boss GIF Images

To display animated GIFs of the bosses, add the following files to this folder:

- `andariel.gif` - Andariel, Maiden of Anguish
- `duriel.gif` - Duriel, Lord of Pain
- `mephisto.gif` - Mephisto, Lord of Hatred
- `diablo.gif` - Diablo, Lord of Terror
- `baal.gif` - Baal, Lord of Destruction

## Where to Find GIFs

You can find Diablo II boss animated GIFs from:
- Diablo Wiki (diablo-archive.fandom.com)
- DeviantArt
- Spriters Resource
- Or create your own from game sprites

The images will automatically load once placed in this folder with the correct filenames.
